用 $\LaTeX$ 排版文字

排版中文

源码统一使用 utf8 编码，用 xelatex 命令编译。源码中使用 ctex 宏包或者文档类。

1
2
3
4
5

\documentclass{article}
\usepackage[UTF8]{ctex}
\begin{document}
你好，这是一个测试文档。
\end{document}

或者

1
2
3
4
5

\documentclass[UTF8]{ctexart}
\usepackage[T1]{fontenc} %为保证某些符号显示正确。
\begin{document}
你好，这是一个测试文档。
\end{document}

这里参考了全面总结如何在 LaTeX 中使用中文 (2020 最新版)1。

字符

空格和分段

空格键和 Tab 键输入的空白字符视为“空格”。连续的若干个空白字符视为一个空格。一行开头的空格忽略不计。

行末的回车视为一个空格;但连续两个回车,也就是空行,会将文字分段。多个空行被视为一个空行。

特殊字符

以下字符有特殊含义，所以需要用命令转义。

  • \# -> #
  • \$ -> $
  • \% -> %
  • \{ -> {
  • \} -> }
  • \_ -> _
  • \^{} -> ^
  • \~{} -> ~
  • \textbackslash -> \

\^{}和\~{}本是用来形成参数重音效果的，这里设置为空参数，就直接输出该字符了。\\被定义成手动换行，所以不能用来转义了。

标点符号

中文标点不用留意。这里只讨论英文。

英文单引号和双引号：

1

``Please press the `x' key.''

连字号和破折号有不同的长度：

1
2
3

daughter-in-law
pages 13--67 % 13到67页
yes---or no? % 破折号

省略号 \ldots。

特殊重音

1

Sch\"onbrunner

得到 Schönbrunner。

斜体

1

Some \emph{emphasized words, including \emph{double-emphasized} words}, are shown here.

斜体中的斜体会变成正体。

\emph{} 不要用在数学公式中。

断行和断页

断行会尽量取在两个单词之间。用 ~ 插入不会断行的空格。通常用在英文人名，图表名称等场景。如 Fig.~2a。

手动断行用 \\[⟨length⟩], 既适用于文本，也适用于表格和公式。参数为垂直间距。

断页 \newpage。
