Latex using vim

2020-12-23
vim
约 226 字 预计阅读 1 分钟 次阅读

文章目录

  • latexmk

Vim is a wonderful editor for latex.

latexmk

latexmk allows only one command to compile latex files.

1
2

latexmk # compile
latexmk -c # clean

An example of latexmk config file is provided here: latexmkrc.

Vimtex

vimtex is a plugin for latex files.

I use coc.nvim as my completion plugin so I also need a coc-vimtex.
