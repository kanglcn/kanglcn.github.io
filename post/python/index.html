argparse

argparse 是python的标准库，为程序提供命令行接口(CLI: Command Line Interface). 这里是官方的教程。 This is a well-known blog for argparse library. 这里 是中文版。

1
2
3

parser.add_argument("arg", help="help")
parser.add_argument("SLC_par", type=argparse.FileType help="help")
parser.add_argument('-m','--method',action='action',help='help')

arg is positional argument and method is optional method.

  • action
  • const: useful when cooperating with action if optional argument is flag.
  • help: a description of this argument.
  • metavar: print an example of this argument. Name of positional argument will be replaced so don’t use it in positional argument.
  • type: a constrain of the argument type.
  • choices: only choice in choices can be chosen.
  • default: default value of this argument. Always use it in optional method.

argparse.FileType is a subclass of io.TextIOBase. Simply speaking, it can be used as the return value of open().

Tips for scientific computing

  1. Try your best to us numpy and scipy.
  2. Try your best to avoid for loop. If for loop can’t be avioded, using numba. (sometimes numba accelerated loop is slower than numpy array manipulation. But using numba is better than using nothing.)
  3. Use np.vectorlize vectorlize your function.
  4. Using pytorch in proper situation.

Tips for scientific computing data store and configuration

  1. Use HDF5 for data sets managing, h5py is a good python wrapper of it.
  2. Use OpenBLAS or MKL as BLAS library.
  3. Use python-
  4. Use cuda for GPU-accelerated computations.
