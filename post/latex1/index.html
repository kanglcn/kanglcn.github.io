$\LaTeX$ 基本概念

$\LaTeX$ 命令和代码结构

$\LaTeX$ 中命令以反斜线\开头，为以下两种形式之一:

  • 反斜线和后面的一串字母，如 \LaTeX。它们以任意非字母符号(空格、数字、标点等)为界限。
  • 反斜线和后面的单个非字母符号，如 \$。

字母形式的命令忽略其后的所有连续空格。如果要人为引入空格,需要在命令后面加一对花括号 {} 或者\<空格> 阻止其忽略空格。

$\LaTeX$ 命令有可选参数（用[]包裹）和非可选参数（用{}包裹）。环境类似。

1
2
3

\begin{⟨environment name⟩}[⟨optional arguments⟩]{⟨mandatory arguments⟩}
...
\end{⟨environment name⟩}

$\LaTeX$ 使用一对花括号 { 和 } 作为分组,在分组中使用的命令被限制在分组内,不会影响到分组外的内容。

宏包和文档类

见上述示例。

用

1

texdoc ⟨pkg-name⟩

获取宏包的帮助。

相关文件

  • .sty。宏包文件。宏包的名称与文件名一致。
  • .cls。文档类文件。文档类名称与文件名一致。
  • .bib。bibtex 参考文献数据库文件。
  • .bst。bibtex 用到的参考文献格式模板。

文件组织方式

1

\include{chapters/a.tex} %相对路径

读入前会另起一页。

1

\input{chapters/a.tex} %相对路径

不会另起一页。
