Here are some key bindings of pllugs I use.

vim-table-mode

vim-table-mode helps writing table in markdown file.

key bindings

key bindings functions
LEADER+tm toggle vim table mode
LEADER+tdd delete the current row
LEADER+tdc delete the current column
LEADER+tic insert the current column
i | text object
a | text object

using

  • Enter the first line, delimiting columns by the | symbol. The plugin reacts by inserting spaces between the text.

  • In the second line (without leaving Insert mode), enter | twice. The plugin will write a properly formatted horizontal line.

  • When you enter the subsequent lines, the plugin will automatically adjust the formatting to match the text you’re entering every time you press.

vim-surround

vim-surround allows manipulate surroundings.

command functions
cs”<q> “hello” -> <q>hello</q>
cst” <q>hello</q> -> “hello”
ysiw<q> hello -> <q>hello</q>
dst <q>hello</q> -> hello
ysiw] hello -> [hello]
ysiw[ hello -> [ hello ]
yss) add () to the entire line

In visual mode, S+surrounding will surround the selected area.

vim-expand-region

vim-expand-region allows you quickly select the closest text object among a group of candidates.

command functions
v expand the visual selection(in visual mode)
_ shrink the visual selection(in visual mode)

vim-visual-multi

vim-visual-multi allows you to edit file using multi-cusor.

Ctrln adds cusors at places of the selected snippets in visual mode or word when not using visual mode.

command functions
Ctrln add cusor at place of same words
Ctrlj/k add cusor vertically
n/N get next/previous occurrence
[/] select next/previous occurrence
q skip current and get next occurrence
Q remove current cursor/selection

You can modify you file (such i,a,I,A) when you select several words in their place.

This plugin brings awesome convenience when coorperated with f.
