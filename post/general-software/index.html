linux 下安装常用软件

本文介绍 linux 下的常用软件的安装。 Arch系发行版的很大一个好处就是 AUR 中有相当多的软件，可以直接一行命令安装。

常用软件

chrome

1

sudo pacman -S google-chrome

网易云音乐

1

sudo pacman -S netease-cloud-music #网易云音乐

WPS Office

1
2
3

sudo pacman -S wps-office #WPS office 套件
sudo pacman -S ttf-wps-fonts #WPS的一些字体
sudo pacman -S wps-office-mui-zh-cn #WPS的中文语言包

WPS 是目前 linux 系统下可用的，兼容 MS Office 最好的，但还是无法做到完全兼容。使用时要对可能存在的风险做好充分准备。

如果希望兼容的话，还是建议使用 Office 365 网页版。

什么？你说 linux 下的 WPS Office 会缺失 Windows 下的字体，这些字体就在你电脑上为什么就用不了呢？赶紧好好想想，该怎么办！

$\LaTeX$

texlive

1
2
3

sudo pacman -S texlive-most
sudo pacman -S texlive-langchinese  #texlive的中文支持
sudo pacman -S texstudio

texstudio 是一款专业的 $LaTeX$ 编辑器，提供语法高亮。

设置texstudio的 default compiler 为 xelatex，这是目前最好用的 $TeX$ 发行版。

texmacs

1

sudo pacman -S texmacs #一个所见即所得的编辑器

Texmacs 是一款跨平台的所见即所的的编辑器，吸取了 tex 的优点，但和 tex 不是一个东西，可以显著提高数学公式的输入体验和速度。

FileZilla

1

sudo pacman -S filezilla #常用的 FTP 客户端

linux 下的 QQ 和微信

linux下是没有好用的腾讯系通讯软件，QQ for linux是官方出的唯一一款运行在 linux 下的通讯软件。但是，它经常崩溃而且脱胎于上古时期的qq，功能极少，不建议使用。

2020-04-12 update: 目前 QQ for linux 已经不再闪退，可以安装使用。

另一个选择是基于 wine 的软件，wine 是运行于 linux 下的 windows 环境模拟工具，本质上是套娃。目前兼容性的问题很大，所以除了已有的很成熟的软件之外，别的都不推荐。关于 wine 可以参考Wine homepage1。

基于 wine 的目前最稳定的 qq 和微信可以在 AUR 中找到

1
2

yay -S deepin-wine-tim
yay -S deepin-wine-wechat

关于 rm

rm 是一个很危险的命令。我的建议是除非需要在别的计算机上运行，否则一律改用trash-cli，trash-cli的语法和rm不一致。或者使用rmtrash，rmtrash是trash-cli的封装版，和rm语法兼容。这两个工具提供垃圾回收机制，确保安全。我建议即使要在别的计算机上运行，也要在测试稳定之后再把rmtrash换成rm。

rmtrash使用 trash-list列出文件，trash-restore restore文件，使用 trash-empty empty trashcan。进一步了解rmtrash可以参考Github:rmtrash2。

下载

1

yay -S rmtrash

也可以将 rm 重命名为 rmtrash

1
2
3
4

# rmtrash
alias rm='rmtrash'
alias rmdir='rmdirtrash'
alias sudo='sudo '

但不要将上述别名放在/etc/profile中，这样没有任何效果，可能是因为会被之后的配置文件覆盖掉。可以放在~/.bashrc中。
